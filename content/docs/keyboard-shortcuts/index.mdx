This is a list of keyboard shortcuts available in ILEditor 2.

| Shortcut | Description | Usage |
|--------------|-------------------|---------------------------|
| Ctrl+S | Save current file | When editing source code. |
| Ctrl+Z | Undo | When editing source code. |
| Ctrl+Shift+Z | Redo | When editing source code. |
| Ctrl+E | Compile | When editing source code. |
| Ctrl+/ | Toggle Comment | When editing source code. Works for RPG (free), C, C++ and CL |
